# SCVR.WebApp

ASP.NET web application developed for a football club management scenario.

## Features

* Home page with club news and information
* League standings page
* Online store
* Ticket purchase system
* User authentication and login
* Admin management area

## Store and Tickets

Users can:

* View products and tickets
* Purchase items and tickets after authentication
* Access different sections of the platform

## Admin Area

Administrators can:

* Manage products
* Manage orders
* Manage tickets
* Manage users

## Technologies Used

* ASP.NET Core MVC
* C#
* SQL Server
* HTML5
* CSS3
* JavaScript

## Project Structure

The application follows the MVC architecture pattern and includes authentication and role-based access for users and administrators.

## How to Run

1. Clone the repository
2. Open the solution in Visual Studio
3. Configure the database connection
4. Run the application

```bash
git clone https://github.com/carlosfontesdev/SCVR-WebApp.git
```
